# 桌面便签

桌面便签是 MyHelper 的一个实用功能，让您可以在桌面上快速记录和查看笔记。

## 基本功能

### 创建便签
- 快捷键：`Alt + N`
- 系统托盘菜单
- 主界面便签按钮

### 编辑功能
- 富文本编辑
- 支持图片粘贴
- 文字格式化
- 链接识别
- 代码高亮

### 便签管理
- 拖拽移动
- 调整大小
- 置顶显示
- 折叠/展开
- 分组整理
